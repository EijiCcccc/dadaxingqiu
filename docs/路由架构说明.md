# 路由架构说明

## 设计原则

| 原则 | 实现方式 |
|------|----------|
| 禁止直接 Navigator | 统一通过 `GlobalRouter` 跳转 |
| 禁止 context 扩展 | 不提供 `context.go()` 等扩展 |
| 路径常量集中 | `core/router/route_path.dart` |
| 分模块自注册 | 各 feature 提供 `registerXxxRoutes(registry)` |
| 参数类型化 | `AppRoute` sealed class，如 `DetailRoute(id: '123')` |
| 统一 GlobalRouter | `GlobalRouter.instance.go/push/pop` |
| 分包不能直接引用跳转 | feature 只依赖 core，通过 `AppRoute` 类型跳转 |

---

## 目录结构

```
core/
  lib/router/
    route_path.dart       # 路径常量
    app_route.dart        # 类型化路由定义
    global_router.dart    # 全局路由单例
    route_registry.dart   # 路由注册表（供 feature 自注册）
    router.dart           # 导出

feature_home/
  lib/
    router/
      home_routes.dart    # 自注册：registerHomeRoutes(registry)
    presentation/
      home/
        home_page.dart
        home_state.dart
        home_providers.dart
    feature_home.dart

feature_detail/
  lib/
    router/
      detail_routes.dart  # 自注册：registerDetailRoutes(registry)
    presentation/
      screens/
        detail_screen.dart
    feature_detail.dart

app/
  lib/router/
    app_router_config.dart  # 收集各 feature 路由，构建 GoRouter
```

---

## 依赖关系

```
app ──► feature_home, feature_detail, core
feature_home ──► core  （不依赖 feature_detail）
feature_detail ──► core  （不依赖 feature_home）
```

---

## 使用方式

### 1. 跳转（任意包内）

```dart
import 'package:core/core.dart';

GlobalRouter.instance.go(const HomeRoute());
GlobalRouter.instance.push(const DetailRoute(id: '123'));
GlobalRouter.instance.pop();
```

### 2. 新增 feature 包

1. 在 `core` 添加 `RoutePath`、`XxxRoute`
2. 创建 feature 包，添加 `router/xxx_routes.dart`：

```dart
void registerXxxRoutes(RouteRegistry registry) {
  registry.add(GoRoute(path: RoutePath.xxx, builder: (_, state) => XxxScreen(...)));
}
```

3. 在 `app_router_config.dart` 添加一行：

```dart
registerXxxRoutes(registry);
```

---

## 自注册流程

```
app 创建 RouteRegistry
    ↓
app 调用各 feature 的 registerXxxRoutes(registry)
    ↓
各 feature 向 registry 添加自己的 GoRoute
    ↓
app 用 registry.routes 构建 GoRouter
```
