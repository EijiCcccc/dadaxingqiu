Design a mobile social app chat conversation screen for an app called "搭搭星球".

Use a modern dark mode UI style similar to WeChat chat interface. 
The design should be clean, minimal, and card-based with rounded message bubbles.

Create the chat conversation interface with three main areas:

------------------------------------------------

TOP NAVIGATION BAR

At the top of the screen include:

- A back arrow icon on the left
- The other user's nickname centered
- A "more" icon (three dots) on the right

Clicking the "more" icon opens a bottom action menu.

The menu contains two options:

举报
取消

The menu should appear as a bottom sheet style panel with rounded top corners.

------------------------------------------------

CHAT MESSAGE AREA

The middle area displays chat messages in a vertical conversation layout.

Each message contains:

- Circular avatar
- Chat bubble
- Message text

Layout rules:

Other user's messages:
Avatar on the left  
Message bubble on the left side

Current user's messages:
Avatar on the right  
Message bubble on the right side

Message bubbles should be rounded rectangles.

------------------------------------------------

SYSTEM MESSAGES

Between chat messages display system messages in the center such as:

时间提示

Example:
12:30

Message recall notifications:

你撤回了一条消息
“对方昵称”撤回了一条消息

System messages should appear centered with smaller grey text.

------------------------------------------------

LONG PRESS MESSAGE ACTIONS

Long pressing a message bubble opens a contextual action menu.

For messages sent by the current user:

Options include:
复制
撤回
删除

For messages sent by the other user:

Options include:
复制
删除

The menu should appear as a small floating action panel near the message bubble.

------------------------------------------------

REPORT FUNCTION

When the user taps "举报" from the top menu:

Display a modal popup dialog.

Popup content:

Title:
举报

Text input area with placeholder:

请输入举报内容

Below the input field place a primary button:

提交

Top right corner of the popup has a close icon.

------------------------------------------------

BOTTOM INPUT AREA

At the bottom of the screen include a message input bar.

Components:

- Text input field
- Send button

Layout:

Input field on the left  
Send button on the right

Button text:
发送

The input bar should stay fixed at the bottom.

------------------------------------------------

VISUAL STYLE

Dark mode interface  
Minimal modern social UI  
Rounded chat bubbles  
Soft shadows  
Clean spacing  
Mobile messaging design similar to WeChat
